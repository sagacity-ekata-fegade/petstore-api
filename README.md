# 🐾 Pet Store API

A simple RESTful API built with **Node.js**, **Express**, and **Prisma** to manage pet data in a MySQL database.

---

## 📦 Tech Stack

- **Node.js**
- **Express.js**
- **Prisma ORM**
- **MySQL**
- **Postman** (for API testing)
